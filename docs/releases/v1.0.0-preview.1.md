# MathLand v1.0.0-preview.1

Godot 4.7.1로 완전히 다시 만든 MathLand의 첫 설치형 미리보기입니다. 휴대폰 세로 화면을 우선하며, 세로 태블릿에서도 같은 구조로 확장됩니다.

## 주요 내용

- 여러 아이 프로필, 4자리 PIN, 프로필별 설정·진도·보상
- 탐험 섬, 오늘의 길, 자유 탐험, 인벤토리, 수집책
- 오답 시 3개 하트 감소와 하트 0 런 종료
- 콤보, 보스 문제, 사과 보상, 모아 캐릭터 반응
- 터치 압축·스프링·햅틱·효과음·파티클을 사용한 즉각적인 버튼 피드백
- 덧셈, 뺄셈, 곱셈, 최소공배수, 소인수분해와 6개 기초수학 활동
- 10틀, 10막대/십진 블록, 수직선, 수 모으기 등 시각 보조자료
- 기본값 OFF인 선택형 적응 난이도와 JSON 기반 현장 조절
- 문제 중 자동재생 없이 스피커 버튼으로만 재생되는 한국어 여성 안내음성
- 오프라인 이벤트 저널, 앱 중단 후 같은 문제 복구, 중복 없는 보상 처리
- 보호자용 공개 데모와 수동 Content Studio

보호자 웹 데모: <https://jinhoofkepco.github.io/Mathland_new2/>

## APK 설치

1. Release의 `MathLand-v1.0.0-preview.1-arm64.apk`와 `.sha256` 파일을 받습니다.
2. Android의 이 출처 앱 설치 허용 안내를 확인합니다.
3. APK를 설치하고 아이 프로필을 새로 만듭니다.

패키지 ID는 `com.jinhoofkepco.mathland`이며 Android 7.0(API 24) 이상 ARM64 기기를 대상으로 합니다.

## 이 미리보기의 경계

- APK는 GitHub에서 바로 시험할 수 있도록 Android debug 인증서로 서명한 미리보기입니다. Play Store용 영구 배포 서명이 아닙니다.
- 운영 Supabase 프로젝트의 URL·publishable key가 제공되지 않아 배포 APK는 오프라인 모드입니다. 따라서 실제 아이 기기 연결과 외부 보호자 계정의 원격 데이터 확인은 이 빌드에서 활성화되지 않습니다.
- GitHub Pages는 실제 아이 데이터가 없는 합성 데모입니다.
- 새 설치이며 기존 SeoaQuiz의 로컬 진도를 가져오지 않습니다.

소스에는 운영 비밀키, refresh token, 아이의 실제 학습기록이 포함되지 않습니다. 운영 클라우드를 연결할 때에는 저장소의 Supabase migrations/functions를 별도 프로젝트에 배포하고 공개 설정만 빌드 staging에 주입해야 합니다.
